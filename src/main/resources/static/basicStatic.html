这是基本静态页面，web发展初期，静态页面可以展示一些固定的内容，单纯的HTML页面，什么也没有，HTML全称为HyperText Markup Language,超文本标记语言

单独的文字、表格太过于枯燥， 那么就出现了CSS，CSS起到了对页面修饰的作用，让原本死气沉沉的页面变得好看、赏心悦目、充满活力起来，也为网页吸引来更多的 用户。
CSS全称Cascading Style Sheet，中文译为层叠样式表，css的使用在<style>标签里, style标签不会像<script<一样，放在body标签中，它是放在head标签中， css的核心操作为选择器 {属性1:值1;属性2:值2}，基本文档 最主要的选择器主要是两个，一个是id，利用#id进行设置；一个是class,利用.class进行设置,更多CSS选择器的使用

CSS为单调的网页添加了各种各样的样式，甚至可以添加一些动画。到这里，这些网页的功能只是展示、供人看。 试想一下，在web初期，面对各种加了CSS的页面，一开始人们的兴趣特别浓烈，因为没见过。但是经过一段时间发展之后，面对各种各样的HTML+CSS页面，会产生疲劳感。 如何让用户看网页的时候不疲劳？在于网页需要和用户产生一定的互动
javascript负责的就是这个，如果说CSS的动画能够让网页动起来一部分，那么Javascript则是完全让用户和网页都动起来,这里的动我更认为是二者之间产生互动， 例如，， 通过JavaScript和HTML的DOM的结合，使得通过JavaScript可以对DOM进行相应或者改变，使得网页和用户的互动率大大提高； 这样一来，就大大增加了用户对于网页的留存率。关于JavaScript和HTML的DOM的相关知识可以参考相关文档和DOM的相关属性和方法

最后说一下HTML、CSS、JavaScript三者之间的关系和联系，从上述之中可以看到，一个网页最基础的地基是HTML，它定义了一个网页该有哪些元素组成，是段落、还是表格，还是标题等等。CSS是使得网页更好看、漂亮 ，而JavaScript使得网页能够【动-和用户产生互动】起来。举一个形象的例子，如果把网页当做一个玩偶，HTML定义了这个玩偶该有什么零件，头、四肢等。CSS则是为这个玩偶人体模型化了妆，创了衣服等等。而JavaScript则可以让这个玩偶和用户产生有趣的互动，而不是仅仅被用来看 借用网上的话来说就是：

  • HTML provides the basic structure of sites, which is enhanced and modified by other technologies like CSS and JavaScript.
  • CSS is used to control presentation, formatting, and layout.
  • JavaScript is used to control the behavior of different elements
